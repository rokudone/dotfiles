# 目的

墨田区の家事支援事業に参画したいと考えています。向こうが興味を持ってくれるようなメール文を作成しなさい。

# 背景

- 墨田区は一般ユーザ向けの家事支援事業を検討している。そこに参画できるのは一社のみである
- 私たちは株式会社CaSy(https://casy.co.jp/)という会社で家事支援サービスを運営しており、その事業に参画できることは大きなビジネスチャンスである
- 選考はメールによってのみ行われるため、メールの内容において他社との差別化を測る必要がある。
- 相手は行政区であり、お堅い文章が求められる。

# 内容

メール本文には以下の内容を含めなさい。

- 挨拶
- 会社紹介
- サービス紹介
- 初回アポイントメントの依頼
- その他、有用であるものがあれば提案しなさい
