# Diff

このプロセスで変更したファイルのみをdiff形式でエディタ表示してください。

## 手順：
1. git diff --name-only でワーキングディレクトリの変更ファイル一覧を表示
2. git diff --cached --name-only でstagedファイル一覧を表示
3. このセッションで実際に編集したファイルのみを抽出
4. 各ファイルをdiff形式でエディタ表示：

## 出力形式：
```
このプロセスで変更したファイル

file1.py
file2.js
config.yaml

diff表示で開きます...
```

## 実行
各ファイルの差分をエディタで表示してレビュー
